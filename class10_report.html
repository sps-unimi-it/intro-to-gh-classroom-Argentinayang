Executive Summary

Based on the emotional data of ANES 2024 voters, this report provides data-driven allocation suggestions for the Biden campaign team to budget $1,000,000 in the last week. By analyzing the emotional thermometer scores of voters for different social groups, we identify three types of key target groups and formulate corresponding budget strategies.

Key Findings: The most popular groups (high thermometer ratings) include Jews, Hispanic, Asian, White, Black, and Rural Americans. - The least popular groups (low thermometer ratings) include KKK, Nazi, White Supremacists, Fascists, Antifa, and Anti-gay. - Persuadable groups (ratings near 50) are prioritized for targeted messaging.

Recommendation: Allocate the budget as follows: - 60% ($600,000) to Persuadable Groups - 30% ($300,000) to Mobilization (High-Liking Groups) - 10% ($100,000) to **Defensive (Low-Liking Groups)

  1. Rate the group according to the “persuasiveness” index (the closer to 50, the higher);

  2. Assuming that the total budget is $1,000,000;

    b.30%:Mobilization (high liking group)

    c.10%:Defensive (low liking group)

# 3. Persuasion groups (60% of budget) - groups closest to 50
persuade_budget <- total_budget * 0.6
# Recalculate persuasion allocation
group_summary <- group_summary[order(-group_summary$persuade_score), ]
group_summary$allocated_persuade <- (group_summary$persuade_score / sum(group_summary$persuade_score, na.rm = TRUE)) * persuade_budget

# Reset mobilization and defensive allocations
group_summary$allocated_mobilize <- 0
group_summary$allocated_defensive <- 0

# Apply allocations

  group_summary$group %in% top_liked_corrected$group,
  mobilize_budget / 5,
  0
)

group_summary$allocated_defensive <- ifelse(
  group_summary$group %in% least_liked_corrected$group,
  defensive_budget / 3,
  0
)

# Recalculate total
group_summary$allocated_total <- rowSums(
  group_summary[, c("allocated_persuade", "allocated_mobilize", "allocated_defensive")],
  na.rm = TRUE
